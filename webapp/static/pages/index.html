The only site able to track your LeetCode learning activity and progress

Install the browser extension LeetPlug (Chrome and Firefox are supported) to enable the main features totally integrated with the LeetCode site.
Disclaimer: this is not a LeetCode product and it is not officially supported by LeetCode site. It is a separated project which uses the data publicy available on the user browser.

Main features:

  • Automatic hiding of the LeetCode problem description: you just need to see them when you are ready
  • Integrated stopwatch: time is critical when solving an interview question. Keep an eye on it while you are solving the problem.
  • Problem submission tracking, along with the time spent to solve a question: continue to improve your skills lowering the time you spent on most difficult questions
  • Last successful timestamp tracking, to be able to refresh the problems which you are not solving since long time

The target of this site is to provide an easy way to LeetCode users to keep track of most difficult questions. If you keep track of the time spent to solve all the questions you can easily sort them and then periodically solve the most critical ones.
Please follow the installation and registration instructions to start to track your progresses!

This is an independent project totally open source. I need your help to improve and to maintain it. If you find it useful please donate and you will help to cover the expenses to keep the site up and running!
